Introduction

On each SAT Math test there are several questions that require the student to solve algebraic expressions. The algebraic expressions range from simple (one step) to complex (several steps). Simple algebraic expressions involve solving simple linear equations that usually have just one variable.

We start this section with a review of the order of operation. PEMDAS, a mnemonic used to remember order of operations, means algebraic expressions are simplified in the following order - Parentheses, Exponents, Multiplication, Division, Addition, and Subtraction.

Simple algebraic expressions require a basic working knowledge of
  • the order of operations
  • isolation of variables
  • factoring a greatest common factor
  • the ability to solve for more than one variable at a time

2.1a PEMDAS and the Order of Operations

We can use the mnemonic PEMDAS to remember the correct order of operations.
  • Parentheses
  • Exponents
  • Multiplication and Division

    (must be completed in the order written in question)

  • Addition and Subtraction

    (in any order)

Notice that multiplication and division, as well as addition and subtraction, are grouped together since this is a common source of student confusion when applying the PEMDAS mnemonic. If multiplication and division are the only operations remaining in an expression, they must be completed in the order in which they are written in the question. Addition and subtraction can be completed in any order.

Example 1
Remember: Multiplication and division must be completed in the order written in the question.

Simplify the following expression: \( 6 ÷ 2 \times 10 \)

  • Must do division first since it appears first in the equation.
  • $$6 ÷ 2 = 3$$ $$3 \times 10=30$$
    • Example 2
    Remember: Addition and subtraction can occur in any order.

    Simplify the expression: \( 4+7-3+8 \)

  • You could rearrange the terms to put all of the additions together \( 4 + 7 + 8 = 19\) and then subtract the 3 from 19 \((19-3)\) to get 16
  • ...or, you could add \(4 + 7\) to get 11, subtract 3 to get 8 and then add 8 to get 16.
  • Either way, the answer is 16.
    • Example 3
    $$14 + 18 ÷ 2 \times 18-7 $$
    • Parentheses: None
    • Exponents: None
    • Division: There is a division; perform that calculation \( 18÷2=9\) $$14 + 9 \times 18-7$$
    • Multiplication: There is a multiplication; perform that calculation \(9 \times 18 = 162\) $$14 + 162 - 7$$
    • Addition/Subtraction: The only remaining calculations are addition and subtraction, these can be done in any order.  The answer is 169. 
    Example 4
    $$ (9 + 33-6) ÷ 6-3^2 $$
    • Parentheses: There is a parenthesis in the expression. Simplify this part of the expression. \((9 + 33-6 = 36)\)$$36 ÷ 6-3^2$$
    • Exponents: There is an exponent. Simplify this part of the expression (\(3^2 = 9\)). $$36÷6 - 9$$
    • Multiplication: None
    • Division: There is a division. Solve this part of the expression ( \(36÷6=6\) ) $$6-9$$
    • Addition: None
    • Subtraction: Last remaining term. Solve this part of the expression to arrive at final answer (\(6-9=-3\)) $$-3$$

    Sample Problem 1
    Remember: PEMDAS should also be used with polynomial expressions.
    $$(6 + 3)^2 + (9-10 ÷ 5)$$
    • Parentheses: There are two parentheses in this expression. Simplify each part independently \((6 + 3 = 9)\). In the second set of parenthesis, we now have to use PEMDAS to simplify. Division comes before subtraction, therefore we have to do that operation first \((10 ÷ 5 = 2)\). Now we have to do the subtraction \((9-2 = 7)\) The expression now looks like this: $$(9)^2 + (7)$$
    • Exponents: There is an exponent. Simplify this part of the expression ( \(9^2 = 81\) ). $$81 + 7$$
    • Addition: Complete the last remaining part of the expression. \((81 + 7 = 88)\) $$88$$
    Sample Problem 2
    Remember: PEMDAS should also be used with polynomial expressions.
    $$ \dfrac{(3x+2x)^2}{x^2}-\dfrac{6x^3}{x^2\times2x} $$
    • Parentheses: There is a parenthesis in this expression. Simplify that part of the expression \(3x + 2x = 5x\)
      $$\dfrac{(5x)^2}{x^2}-\dfrac{6x^3}{x^2\times2x} $$
    • Exponent: There is an exponent in this expression. Simplify that part of the expression. (\(3x+2x=5x)\) $$(5x)^2 = 5x \times 5x = 25x^2$$ $$ \dfrac{25x^2}{x^2}-\dfrac{6x^3}{x^2\times2x} $$
    • Multiplication: There is a multiplication in the expression. Simplify that part of the expression. \( (x^2 \times 2x = 2x^3) \)$$ \dfrac{25x^2}{x^2}-\dfrac{6x^3}{2x^3}$$
    • Division: There is a division; perform that calculation \((18 ÷ 36 = \frac{1}{2})\)$$ 14 + \frac{1}{2}-7 $$
    • Division: There are two separate divisions in the expression. Simplify that part of the expression. \( (25x^2 ÷ x^2 = 25) \text{ and } (6x^3 ÷ 2x^3 = 3) \)The expression should now look like: $$25-3$$ Solve the remaining subtraction to get 22 as the final answer.
    Sample Problem 3
    Remember: PEMDAS should also be used with polynomial expressions.
    $$ 3+\dfrac{4(x-2)^2}{x-2}+\dfrac{x^2}{x}-11x+\dfrac{18x^2}{3x}$$
    • Parentheses: There are parentheses in the expression, but they cannot be simplified.
    • Exponent: There is an exponent, but we want to wait to expand this until we determine if it can be simplified.
    • Multiplication: There is a multiplication, but it is connected to something in a parenthesis and therefore, we are going to wait to simplify.
    • Division: There are multiple divisions in this expression. Simplify each part of the expression independently.
      $$\dfrac{4(x-2)^2}{(x-2)} = 4(x-2)$$ $$\dfrac{x^2}{x} = x$$ $$\dfrac{18x^2}{3x} = 6x$$ The expression should now look like this:
      $$3 + 4(x-2) + x-11x + 6x$$
      Multiplication revisited: Let distribute the 4 outside the parenthesis to give \(4(x-2) = 4x-8\) $$3 + 4x-8 + x-11x + 6x$$
      Combine like terms
      $$4x + x-11x + 6x = 0$$ $$3-8 = -5$$ Therefore the final answer is -5.
    Sample Problem 4
    Remember: PEMDAS should also be used with polynomial expressions.

    $$\dfrac{1}{(x+3)^3\times(x+3)^4}-\dfrac{(x-3)^2}{(x-3)}$$

    • Parentheses and Exponents: Although these are present in the expression, we typically can’t simplify parentheses and want to wait to expand any exponents until we determine the simplest possible exponent. The exponent may be reduced as a result of multiplication and division.
    • Multiplication: There is a multiplication in this expression. This multiplication is easier to see if we rewrite it to look more mathematical.$$\dfrac{1}{(x + 3)^3} \times (x + 3)^4 = (x + 3)^1 = (x + 3)$$
    • Division: There is a division in this expression. Simplify this part of the expression. $$\dfrac{(x-3)^2}{x-3} = (x-3)$$ The expression should now look like this: $$(x + 3)-(x-3)$$ Distribute the - sign into the second parenthesis to give: $$x + 3-x + 3$$ Combine like terms to give the final answer: 6

    2.1b Common Mistakes

    There are few operations that cause students to miss questions they would otherwise answer correctly:
    • incorrectly applying order of operations
    • failing to distribute negative 1 (a negative sign)
    • not recognizing when to factor out a variable when there are other variables present
    • the inability to solve for two or more variables as a whole.

    This material is best presented through a series of examples.
    Example 9
    Incorrectly Applying Order of Operations
    This is a common source of error in more complex looking mathematical expressions.

    Simplify the expression: \( 4-7(x-3) \)

    Error
    • \(4-7(x-3) \)
    • \(-3(x-3) \)
    • \(-3x + 9\)
    Correct
    • \(4-7(x-3) \)
      • distribute -7 into ( )
    • \(4-7x + 21\)
      • final, correct answer
    • \(25-7x\)
    Example 10

    Solve the expression: if \(y=3x^2\) and \(x=2\), what is the value of \(y\)?

    Error
    • \(y=3(2)^2\)
    • \(y=6^2\)
    • \(y=36\)
    Correct
    • \(y=3(2)^2 \)
    • \(y=3(4)\)
      • final, correct answer
    • \(y=12\)
    Example 11

    Simplify the following expression: \( 10(x-4)-(3x-20) \)

    Whenever there are parentheses in an expression, we must first decide if they are necessary. That is, is there anything to distribute to the terms inside the parentheses?

    In this case there is a 10 outside the first set of parentheses that needs to be distributed and a negative 1 outside the second set of parentheses that needs to be distributed to both terms. Distribution of both of these terms gives the following result.

  • \(10x-40-3x+20\) Distribution
  • \(10x-3x-40+20\) Rearranging to get like terms next to each other
  • \(7x-20\) Combine like terms
    • Example 12

    What is the value of \(a+b\)? $$4a-7+4b=17$$

  • Rearrange the equation to get \(a\) and \(b\) next to each other $$4a+4b-7=17$$
  • Get the \(a\) and \(b\) terms by themselves $$4a+4b=24$$
  • Divide the entire equation by 4 $$a+b=6$$
  • The sum of \(a\) and \(b\) is 6.

    • Alternatively, we could just choose a value for one of the two variables, solve for the other and complete the requested mathematical operation. Let \(a = 2\). Plug 2 into the equation for \(b\).

  • $$4(2)-7+4b=17 $$
  • $$8-7+4b=17$$
  • $$1+4b=17$$
  • $$b=4$$
    • Therefore, \(a+b = 2+4=6\) This method will give the correct answer regardless of the value chosen for \(a\)
    Example 13

    Solve for \(y\): $$4y+xy=8z-7$$

    Since \(y\) appears in two terms, one of which contains an \(x\), there is no way to add those two terms together. Thus, we need to factor out the \(y\) in order to get \(y\) by itself. Finally, dividing by \(4 + x\) gives the final answer.

  • Factor out \(y\) $$y(4+x)=8z-7$$
  • Divide by \(4+x\) $$y=\dfrac{8z-7}{4+x}$$
